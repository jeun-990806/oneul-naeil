# 오늘내일

## 1. 소개
 밀린 일을 처리해야 할 때, 할 일들을 쭉 나열해두고 하나씩 처리하는 것도 좋은 방법입니다. 그런데 펜과 종이가 없을 때에는 어떻게 해야 할까요? 모니터 한 대만으로, 할 일들을 옆에 항상 띄워둘 수 있다면 좋지 않을까요? '오늘내일'은 그러한 생각으로부터 만들어진 애플리케이션입니다. 간결한 기능과 UI를 통해 '오늘 내 일'들을 하나씩 해치워 봅시다.
